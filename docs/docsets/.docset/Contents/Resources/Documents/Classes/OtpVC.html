  • didChangeVerificationCode()

    Notifies that the text in KWVerificationCodeView has been changed. This is especially useful in situations where you have to enable the submit button only if the verification code is valid. If entered verification code is valid, set verify button to enabled, its alpha value to 1, resend button as disabled and its alpha to 0.5, else verify button to disabled, its alpha value to 0.5 and resend button to enabled and its alpha value to 1
